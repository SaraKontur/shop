# 🛒 Simple Shop App (Android)

A robust native Android e-commerce application built with **Kotlin**. This project demonstrates modern Android development practices, local data persistence, and role-based user management.

## ✨ Features

### 👤 For Users
* **Authentication:** Sign up and Log in system.
* **Product Browsing:** Search bar, category filters, and detailed product views.
* **Shopping Cart:** Add items, adjust quantities, and **swipe-to-delete** functionality.
* **Favorites:** Save products for later.
* **Order History:** View past orders with detailed status (Created, Shipping, Delivered).
* **Reviews:** Leave ratings and text reviews (with photo/video support).
* **Profile:** Manage avatar and personal details.

### 🛡️ For Admins
* **Product Management:** Create, edit, and delete products.
* **Category Management:** Add or remove product categories dynamically.
* **Order Management:** View all user orders.
* **Role-Based UI:** Special buttons and controls visible only to admins.

## 🛠 Tech Stack

* **Language:** Kotlin
* **Architecture:** MVVM (Model-View-ViewModel) pattern concepts
* **Database:** Room (SQLite) for local data storage
* **UI:** XML Layouts, Material Design Components
* **Concurrency:** Coroutines (for database operations)
* **Images:** URI handling and local storage
